# Bibliography

This file lists every third-party scholarly work the project draws on. The PDFs themselves live under `references/` in the **private companion repository** (`minoan-linear-a-references`) — see this repo's README for the rationale. Local paths in this file (`references/core/...pdf`) point into that companion repo, which is sibling to this one on the maintainer's machine and accessed via `scripts/_paths.py`. Each entry also lists, where known, a public source URL so anyone reading this file can locate the original.

Categories used:

- **CORE** — mainstream Linear A scholarship that the project relies on for facts and consensus positions.
- **HETERODOX** — work by serious researchers advancing positions outside mainstream consensus, but engaging substantively with the corpus. Currently dominated by Duccio Chiapello's "Minoan = Greek dialect" series.
- **ANCILLARY** — decipherment claims not accepted by the field; retained to document the landscape and to engage with specific arguments where useful.
- **COMPARATIVE** — work on adjacent scripts, languages, or material cultures (Hurrian, Cuneiform, alphabetic, Cypro-Minoan, Cypriot syllabic, etc.).
- **BIBLIOGRAPHIC** — reviews, supplements, indices, edited-volume tables of contents.
- **FUTURE WORK** — Linear B / C / D / Cypriot syllabic material reserved for a later phase.
- **OFF-TOPIC** — material acquired incidentally (e.g. from academia.edu "similar papers" feeds) that turned out to be unrelated. Listed here for transparency, not for analytical use.

Acquisition / holding status labels used in this bibliography:

- **HELD** — clearly present in local holdings under `references/` / `future_work/`
- **MISSING** — cited by works in the corpus but not clearly present in local holdings yet
- **PARTIAL** — metadata, review, excerpt, related article, or uncertain edition present, but not a clean held target
- **AMBIGUOUS** — citation exists but exact bibliographic target still needs disambiguation

Importance labels used for missing / follow-up entries:

- **HIGH** — heavily cited or foundational; should be prioritized for reconciliation/acquisition
- **MEDIUM** — useful but not immediately blocking
- **LOW** — background / context / opportunistic

Where filenames are abbreviated (academia.edu artifacts), the local file is named the same as the original download. Author/year fields are best-effort reconstructions from filenames and first-page extracts; entries marked "(verify)" still need confirmation against the title page.

## HETERODOX — Duccio Chiapello "Minoan = Greek" corpus

A connected sequence of ~22 papers arguing Linear A encodes a Greek dialect. Heterodox: not accepted by mainstream consensus but methodologically engaged with the corpus, published serially. Treated as a coherent body to be analyzed and rebutted (or selectively engaged with) on its own terms. See `docs/analysis/2026-04-13-intake-analysis.md` §3 for the synthesis and `docs/analysis/chiapello-minoan-greek-response.md` (stub) for the project's consolidated response.

## COMPARATIVE — adjacent scripts, languages, and material cultures

### Hurrian hypothesis (van Soesbergen and respondents)

A large body of work by P. Reijnier J. van Soesbergen arguing Linear A encodes Hurrian, plus exchange with respondents (Fournet, Solca). The corpus is collected in `references/comparative/van_soesbergen_hurrian/` — multiple volumes of *The Decipherment of Minoan Linear A* (~7 PDF copies, some duplicates), *Minoan Linear A Vol I: Hurrians and Hurrian* (2 copies), the *Text* volume, lecture notes (Prayers/Libation Tables 2–3, Hurrian Myths and Hesiod 4), and short pieces (appendices on a-pi KN Ze 16 / po-to-ku-ro, comments and replies on Fournet 2018 ×2 and Solca 2018 ×3, letter to Roland Cash on hypocoristic -ya, personal homage to Ilse Wegner, review of Tardivo–Kitselis pre-Greek substrate).
